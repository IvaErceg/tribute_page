Amazing Grace Hopper

Did you know?

First compiler

Grace developed an innovative program that would translate the programmer's language into machine language. This first compiler, called "A-O, " allowed the programmer to write in a higher-level symbolic language, without having to worry about the tedious binary language of endless numbers that were needed to communicate with the machine itself.

Rear admiral

She became captain on the retired list of the Naval Reserve in 1973 and commodore in 1983. In 1985 she earned the rank of rear admiral before retiring in 1986.

COBOL

In 1959 a consortium was formed with the purpose of developing a standard programming language that could be used universally on any computer. Grace Hopper was asked to serve as the technical consultant on the committee, which later developed the programming language COBOL. COBOL, which stands for “COmmon Business-Oriented Language”, is still used in order-processing business software today.

Bug

While working on Mark II computer, she and her colleagues found a moth stuck in one of the electrical switches controlling a circuit which caused error in the program. They literally removed a bug!
